# Architecture Decision Records

ADRs 0001–0010 are the **initial set** — every architectural decision made during the planning conversation that produced this project. They were written before any tools were installed, because CLAUDE.md mandates that any decision constraining future work has an ADR, and a stack of decisions already existed by the time setup began.

**Date convention**: ADRs 0001–0010 share a single date because they were decided in one planning session. Future ADRs use the date of the actual decision.

**Status convention**: All entries here are `accepted`. To supersede one, change its status to `superseded by ADR-NNNN` and create a new ADR.
